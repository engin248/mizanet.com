## TEMEL FELSEFE

```
Sistem bir işletmenin 4 eksenini izler:

TREND → TASARIM → ÜRETİM → SATIŞ

Bu 4 eksenden her biri için farklı veri kaynağı,
farklı kriter listesi gerekir.

Tek noktadan veri yeterli değil.
```

## EKSEN 1: TREND (Ne üretilecek?)

### Platform: TRENDYOL
Her ürün sayfasında DOĞRUDAN OKUNACAK, AYIKLANACAK VE ANALİZ EDİLECEK kriterler (Özet Çıkarılmadan, Ham Veriyle Hüküm Verilecek):
```
1. Ürün adı
2. Marka / satıcı
3. Fiyat (güncel)
4. İndirim oranı
5. Değerlendirme puanı (Örn: 4.8 Yıldız)
6. Gelişmiş Yorum Analizi (SADECE SAYI DEĞİL):
   a. Toplam Yorum Sayısı
   b. Fotoğraflı Yorum Sayısı (Müşteri cidden ürünü göstermek istemiş mi?)
   c. 5 Yıldızlı Yorum Oranı (Genel memnuniyet)
   d. 1 ve 2 Yıldızlı Yorum Teması ("Kırmızı renk soluk geldi", "Kalıbı dar" vb. şikayetlerin anahtar kelimeleri)
7. Satış adedi ("X kez satıldı" rozeti)
8. Favoriye ekleme / Beğeni sayısı (Sepette bekletme iştahı)
9. Ana kategori + alt kategori
10. Kumaş / materyal içeriği
11. Renkler Arası Performans (Önemli!):
    a. Piyasada en çok hangi rengi favoriye alınmış?
    b. Yorumlarda spesifik bir rengin hatası/başarısı konuşuluyor mu? ("Siyahı harika ama kırmızısı iç gösteriyor" gibi veriler)
12. Beden seçenekleri + tükenen bedenler (En kârlı/giden beden tahmini)
13. Kargo hızı (Hızlı tüketim ürünü satıcısı mı?)
14. Trend etiketi ("Trendyol Bu Ürünü Seviyor" rozeti)
15. Benzer ürün sayısı (Rekabet havuzu geniş mi?)
16. Ürün fotoğrafı (URL)
17. Ürün açıklaması (İlk 300 karakter)
18. İade politikasındaki gizli ibareler
```
**❓ Eksik veya yanlış kriter var mı?**

---

### Platform: INSTAGRAM
```
1. Hashtag'in toplam kullanım sayısı (#elbisetrendy vb.)
2. Son 24 saatte hashtag artışı
3. En çok beğeni alan 5 gönderi
4. Video / Reels izlenme sayısı
5. Bahsedilen ürün veya marka
6. Lokasyon (Türkiye içi mi?)
7. Etkileyici (influencer) paylaşımı mı?
8. Ürün etiketi (shopping tag) var mı?
```
**❓ Instagram için başka kriter?**

---

### Platform: TİKTOK
```
1. Video izlenme sayısı
2. Beğeni / yorum oranı
3. Ürün adı mention sayısı
4. Viral ses / müzik kullanan trend
5. "Bunu aldım, beğendim" türü review video sayısı
6. Takipçi sayısı (küçük hesap büyük viral = erken trend)
```
**❓ TikTok için başka kriter?**

---

### Platform: GOOGLE TRENDS
```
1. Arama hacmi (aylık)
2. Hacim trendi (yükseliyor / düşüyor)
3. Coğrafi yoğunluk (İstanbul, Ankara vb.)
4. İlgili aramalar ("kadın elbise" → "yazlık kadın elbise")
5. Sezonluk desen (her yıl bu dönemde mi yükseliyor?)
```

### Platform: ZARA / H&M / MANGO (Rakip izleme)
```
1. Yeni koleksiyona eklenen ürün sayısı
2. Fiyat aralığı
3. Hangi renk öne çıkıyor
4. Hangi materyal kullanılıyor
5. İndirim var mı (eski koleksiyonu boşaltıyor mu?)
```

## EKSEN 2: TASARIM (Nasıl görünecek?)

### Platform: PİNTEREST
```
1. Trend board'lardaki renk paleti
2. En çok kaydet alan desen
3. Siluet trendi (bol/dar, uzun/kısa)
4. Detay trendi (volanlı, apoletli, bölünmüş yaka vb.)
5. Sezon rengi (Pantone trendi ile örtüşüyor mu?)
```

### İç Kaynak: KENDİ ARŞİVİMİZ (Kumaş & Aksesuar)
```
Tasarımcı çalışırken sisteme sorar:
→ "Kırmızı pamuklu kumaşımız var mı?"
→ "Altın düğmemiz kaç adet?"
→ Sistemden görsel + stok bilgisi döner
```

---

## EKSEN 3: ÜRETİM (Nasıl yapılacak?)

### Kaynak: KENDİ SİSTEMİMİZ
```
1. Hangi model üretimde?
2. Hangi aşamada? (Kalıp / Kesim / Dikiş / Bitti)
3. Kaç gün gecikme var?
4. Fire / hata oranı
5. Personel hangi banta atanmış?
6. Günlük üretim adedi vs hedef
7. Makine arıza var mı?
```

### Kaynak: TEDARİKÇİ
```
1. Kumaş fiyatı değişti mi?
2. Stokta ne kadar kaldı?
3. Teslimat tarihi?
4. Minimum sipariş miktarı?
```

---

## EKSEN 4: SATIŞ (Ne kadar sattı?)

### Platform: TRENDYOL SATICI PANELİ (Kendi hesabımız)
```
1. Günlük sipariş adet
2. Günlük ciro
3. İade sayısı / oranı
4. En çok satan model
5. En çok iade edilen model (→ tasarım sorunu işareti)
6. Reklam harcaması vs sipariş
7. Müşteri yorumları (olumlu/olumsuz tema analizi)
8. Stok tükenme hızı
```

### Kaynak: KENDİ SİSTEMİMİZ (Kasa / Siparişler)
```
1. Nakit akışı (giren / çıkan)
2. Ödeme bekleyen siparişler
3. Teslim edilen siparişler
4. Kâr marjı (satış - maliyet)
```
