# Otonom Düzeltme Aşamasında Kodlayarak Onaracağım (ÇOK İYİ OLDUĞUM) İşlemler

| Düzeltme Alanı | Otonom Yazabileceğim Kod Tamiri |
| :--- | :--- |
| **Supabase (F, W, NN)** | Tablolara SQL ile kolon ekler/siler, veri tiplerini onarırım. |
| **Next.js & React (J, L, S)** | Sayfadaki `useEffect` / `useState` döngü hatalarını baştan yazar, hızlandırırım. |
| **Güvenlik Çelik Kapısı (R, AA)** | Sayfalara "Admin Değilsen Girme" `if/else` bloklarını direkt yazarım. |
| **Çöp Temizliği (MM)** | Dosyalardaki tüm `console.log` ve ölü fonksiyonları temizlerim. |
| **Hata Sınırları (Q, X, U)** | Formlara "Sıfırdan küçük giremezsin" ve Try/Catch korumalarını kodlarım. |
| **Para Matematiği (GG)** | `Math.round()` ile kuruş sızdıran hesap formüllerini sıfırdan yazarım. |
| **Önbellek (K, FF)** | N+1 atan API'leri birleştirir, Supabase Channel kodlarını yazarım. |
