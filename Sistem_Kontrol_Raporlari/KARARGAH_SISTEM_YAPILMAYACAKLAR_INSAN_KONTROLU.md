# Otonom Olarak YAPAMAYACAĞIM (Diğer Ekiplere Kuralları Test Edecekleri Liste)

Bu dosya, Antigravity'nin yapay zekasının yetmediği, karar veremediği ve İnsan İradesinin (veya başka bir AI'ın) işletme gözüyle test etmesi gereken 26 kriterdir.

Ben bu 26 kriteri HİÇBİR ZAMAN test etmem ve dokunmam.

| Kriter Kodu | Kriter Adı (Diğer Ekibin Kontrol Edeceği) | Neden Ben Yapmam? |
| :--- | :--- | :--- |
| **A** | Gereklilik | Sayfanın önemine işletme/siz karar verir |
| **B** | Eksik/Fazla | İşletmenin görsel/bilgi ihtiyacını bilemem |
| **C** | Verimlilik | İnsan algısı olan tıklama yorgunluğunu bilemem |
| **D** | İşletme Faydası | Ticari kar/zarar hesabı işletme/insan işidir |
| **E** | Bilgi Türü/Kalitesi | Sunumun/Tablonun şıklığı UX ajanı işidir |
| **G** | Sayfa/Sekme Düzeni | Mobil görsel/renk kararları UX ajanı işidir |
| **H** | Soru Mantığı | Sorunun akışı/Türkçesi metin yazarının işidir |
| **I** | Soru Kalitesi | İnsanın yanlış anlayacağı formatları UX bilir |
| **M** | Fiyat/Maliyet | Sunucu aylık fiili bütçesi Muhasebenin işidir |
| **N** | Agent/Otomasyon | İşçinin buna ihtiyacı olup olmadığını siz bilirsiniz|
| **O** | Sürdürülebilirlik | Şirketin 5 yıllık büyüme kararları şirketindir |
| **T** | Kullanıcı Deneyimi (UX) | "Sezgisel" olma konusu tamamen insan bakışıdır|
| **V** | ERP Benchmark | SAP gibi devlerle rekabeti işletme belirler |
| **Y** | Kapasite/Büyüme Uygunluğu| Eş zamanlı Atölye çalışan sayısını donanım belirler|
| **DD** | Bildirim/Uyarı Sistemi | Mesajın sertlik derecesi İnsan Kaynakları işidir |
| **HH** | Kullanıcı Onboarding | İşçinin AR/TR kullanım anlayışını atölye bilir |
| **II** | Test/Geri Alınabilirlik | Gerçek hayat testleri insanların denemesiyle anlaşılır|
| **KK** | Zaman/Para Birimi | Görsel olarak TL veya $ estetiği UX ajanındadır |
| **QQ** | Veri Taşınabilirliği | Şirketin Supabase'den ayrılma/CSV Alma kararı sizindir|
| **RR** | Görsel Tutarlılık | Font, boşluk, yeşil/kırmızı tonlamaları tasarım işidir|
| **SS** | Konfigürasyon Yönetim | Butonların konum veya kolaylığı arayüz işidir |
| **UU** | Altyapı Teknolojisi | Kredi kartı girilen Vercel tier kararları size aittir |
| **WW** | KVKK / Kişisel Veri Koruma | Türkiye hukuk kuralları bir Hukuk uzmanı/Ajanı gerektirir|
| **XX** | Saha Gerçekliği | Fiziksel kumaş sayımıyla tablet tutmuyor, ben göremem|
| **YY** | İşçi Direnci | İşçi tableti kırar mı, sevdi mi ben bilemem |
